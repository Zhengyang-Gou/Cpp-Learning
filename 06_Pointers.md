# 06 - C++ 指针 (Pointers)

## 1. 基本概念 (Basic Concepts)

在 C++ 中，每一个变量都存储在内存中的某个特定位置，这个位置被称为 **内存地址**。指针变量就是专门用来存放这些地址的。

| 概念名称 | 定义 | 比喻 |
| :--- | :--- | :--- |
| **变量 (Variable)** | 存储具体数值的容器 | 房间里的居民 |
| **地址 (Address)** | 变量在内存中的位置编号 | 房间的门牌号 |
| **指针 (Pointer)** | 存储地址的特殊变量 | 记录了门牌号的纸条 |

## 2. 指针的核心操作符 (Key Operators)

理解指针的关键在于区分 `&`（取地址）和 `*`（解引用）这两个符号。

| 操作符 | 名称 | 作用 | 示例 |
| :--- | :--- | :--- | :--- |
| **`&`** | **取地址符 (Address-of)** | 获取一个现有变量的内存地址 | `&var` 得到 var 的地址 |
| **`*`** | **解引用符 (Dereference)** | 通过地址找到对应的变量，并操作它的值 | `*ptr` 访问 ptr 指向的内容 |

## 3. 指针的内存逻辑

当你执行 `int* ptr = &var;` 时，发生了以下过程：
1. **申请空间**：系统为指针 `ptr` 分配了 8 字节（64位系统）的内存。
2. **存入地址**：将变量 `var` 的十六进制地址（如 `0x61ff08`）存入 `ptr` 的内存空间中。
3. **间接访问**：通过 `*ptr`，程序不再看指针本身的值，而是顺着地址去找目标变量。

---

## 示例代码

```cpp
#include <iostream>

int main()
{
    // 1. 定义一个普通变量并赋值
    int var = 8;
    
    // 2. 定义指针并初始化
    // int* 代表这是一个指向 int 类型的指针
    // &var 获取 var 的内存地址并赋值给 ptr
    int* ptr = &var; 

    // 3. 打印指针信息
    // std::cout << ptr;    // 输出的是类似 0x7ffee... 的地址
    // std::cout << &var;   // 输出结果与上面一致

    // 4. 解引用修改值 (Dereferencing)
    // *ptr = 10 意味着：去 ptr 存的地址那里，把那里的内容改为 10
    *ptr = 10; 

    // 此时 var 的值已经被间接修改了
    std::cout << "var 的当前值为: " << var << std::endl; // 输出 10
    
    return 0;
}
